## github-changelog-generator

Generate CHANGELOG/HISTORY markdown files for your project

```yaml
...
    steps:
      - uses: actions/checkout@v2
      - name: Generate CHANGELOG
        uses: tj-actions/github-changelog-generator@v1.15
        with:
          output: 'HISTORY.md'
```
